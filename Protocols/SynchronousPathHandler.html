SynchronousPathHandler

public protocol SynchronousPathHandler : PathHandler

A PathHandler that can handle paths synchronously

  • handle(path:)

    Attempt to handle the provided path. The path can be any type, but the handler does not have to attempt to handle all path types.

    If the provided path cannot be handled the implementation must return nil. If the provided path has been handled the implementation return the path handler that handled the path.

    Return Value
